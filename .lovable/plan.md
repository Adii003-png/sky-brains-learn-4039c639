
# AI in Aerospace Engineering — Portfolio Website for Aditya & Divyanshu

## Overview
A bold, colorful, and highly interactive portfolio website showcasing an AI-powered aircraft maintenance prediction project by **Aditya** and **Divyanshu**. The site will educate visitors about aerospace engineering, demonstrate how machine learning is applied to predict aircraft engine life, and feature rich animations and interactivity throughout.

---

## Pages & Sections

### 1. Hero Section
- Bold, eye-catching hero with an animated aerospace-themed background (gradient + floating particle/plane elements)
- Title: "AI Meets Aerospace Engineering" with animated text reveal
- Tagline introducing the project by Aditya & Divyanshu
- CTA buttons: "Explore the Project" (scrolls down) and "About Aerospace Engineering" (links to section)

### 2. What is Aerospace Engineering?
- Interactive accordion/tab-based section explaining aerospace engineering disciplines (aerodynamics, propulsion, structures, avionics, etc.)
- Each topic has an icon, short description, and a "Learn More" expandable card with animations
- Animated illustrations/icons that respond to scroll

### 3. AI in Aerospace — Why It Matters
- Animated scroll-based storytelling section explaining how AI/ML is transforming aerospace
- Key use cases: predictive maintenance, flight optimization, autonomous systems, materials analysis
- Each use case appears with a fade-in animation as the user scrolls

### 4. The Project — Aircraft Engine Life Prediction
- Detailed showcase of the project (inspired by the AirML repository)
- **How it works** — 3-step animated flowchart: Training → Backend → Frontend
- **Dataset** — Interactive info card about NASA's Turbofan Engine Degradation Simulation Dataset (249 engines, 21 sensors, 3 operational settings)
- **ML Models** — Comparison cards for Random Forest, XGBoost, and Neural Network with interactive toggle to see which performed best
- **Interactive Charts** — Recharts-powered visualizations showing:
  - Sample sensor data trends
  - Model accuracy comparison bar chart
  - Engine remaining useful life predictions

### 5. Interactive Quiz — Test Your Aerospace Knowledge
- A fun, multi-question quiz about aerospace engineering and AI
- Questions with multiple-choice answers
- Score reveal at the end with animated results
- All buttons fully functional and linked to scoring logic

### 6. Meet the Team
- Profile cards for **Aditya** and **Divyanshu**
- Hover animations on cards
- Roles and contributions to the project

### 7. Tech Stack & Architecture
- Visual representation of the technologies used (Machine Learning, Django backend, Angular frontend — presented conceptually)
- Animated flowchart showing data flow from sensor readings → ML model → predictions → dashboard

### 8. Footer
- Contact section with clickable links
- Social/GitHub links
- Smooth scroll-to-top button

---

## Design & Interactivity
- **Bold & Colorful** palette: deep blues, vibrant oranges/teals, gradient backgrounds
- **Scroll-triggered animations** on every section (fade-in, scale-in, slide effects)
- **Hover effects** on all cards and buttons
- **All buttons are functional** — navigation, quiz interactions, section links, expand/collapse
- **Interactive data visualizations** using Recharts
- **Smooth scrolling** navigation between sections
- **Responsive design** for mobile and desktop
